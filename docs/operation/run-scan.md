# Running a Scan

This page describes the recommended flight workflow.

---

## Workflow (4 steps)

1. Place the **QR-code fiducial marker**
2. Initialize coordinate frame in the phone app
3. Take off and start autonomous scanning from the controller
4. Monitor in viewer; end mission; replay the session

---

## During flight

- Always keep manual takeover ready
- Pause/land if tracking becomes unstable
- Watch battery threshold behavior

---

## End mission

- Confirm session is saved
- Replay in viewer and verify outputs
