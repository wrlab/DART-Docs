# Data Export

SkyMapper outputs are useful for downstream tasks:

- facility documentation
- inspection annotations
- ML perception pipelines (e.g., defect detection)

---

## Export checklist

- [ ] session folder is complete
- [ ] metadata included (device models, versions)
- [ ] coordinate frame definition documented

---

## Recommended export formats

- Trajectory: CSV
- Video: MP4
- Reconstruction: mesh/voxel output (document exact format in Reference → File Formats)
