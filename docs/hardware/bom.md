# Bill of Materials (BOM)

!!! note "Start conservative"
    Begin with a known-good baseline configuration and only then swap parts.

---

## Core components

- Drone (SDK-capable)
- Smartphone (AR-capable)
- Mount (3D printed)
- Fasteners / straps (secure phone)
- Optional damping pads

---

## Reference build (tested baseline)

- Drone: **DJI Phantom 4 Pro V2.0**
- Phone: **iPhone 15 Pro**
- Mount: 3D-printed adapter + straps

---

## Tools

- Hex keys / screwdrivers
- Calipers (optional but helpful)
- Threadlocker (low strength, optional)

---

## Nice-to-have

- Prop guards
- Spare props
- Dedicated Wi‑Fi router/AP
